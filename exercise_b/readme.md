# HOW TO RUN

1. Ensure you have Python 3 installed.
2. Open a terminal and navigate to the project root `ejercitacion_b`.
3. Execute the main script `python3 src/main.py`.
4. After execution, the following files will be generated in `data/output`:
    - **episodes_clean.csv**: The cleaned and deduplicated episode catalog from `catalog.csv`.
    - **report.md**: A data quality report with statistics.
